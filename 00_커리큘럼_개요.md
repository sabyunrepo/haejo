# 비개발자를 위한 AI 서비스 만들기 — 2일 완성 커리큘럼

> **부제**: 코드를 한 줄도 못 써도 괜찮아요. Claude Code가 같이 만들어 줍니다.

---

## 1. 강의 한 줄 요약

**Claude Code Desktop**(AI 코딩 도우미)에게 말로 시켜서, **Next.js**(웹사이트 뼈대) + **Supabase**(데이터 저장소·로그인) + **Gemini API**(무료 AI 기능) + **Vercel**(인터넷 공개)로 **나만의 AI 서비스**를 직접 만들어 인터넷에 배포해 봅니다.

---

## 2. 학습 목표

강의를 마치면 수강생은 다음을 할 수 있습니다.

1. AI 시대의 "서비스 한 개"가 어떻게 구성되는지 머릿속에 그림을 그릴 수 있다.
2. 윈도우 PC에 개발에 필요한 도구를 직접 설치할 수 있다.
3. Claude Code Desktop에게 한국어로 요청해서 웹사이트 코드를 받아낼 수 있다.
4. Supabase로 회원가입/로그인과 내 데이터 저장 기능을 만들 수 있다.
5. 무료 Gemini API를 연결해서 "AI 요약" 같은 AI 기능을 추가할 수 있다.
6. Vercel을 통해 만든 서비스를 누구나 접속 가능한 URL로 공개할 수 있다.
7. 위 패턴을 응용해 본인이 원하는 다른 서비스(일기, 쇼핑몰 메모, 독서록 등)도 만들 수 있다.

### 4-1. 하드웨어 / 환경

| 항목 | 권장 사양 |
|---|---|
| OS | **Windows 10 / 11** (강의 기준) |

| 네트워크 | 안정적인 인터넷 (다운로드·배포 시 필수) |

### 4-2. 미리 만들어 두면 좋은 계정 (모두 무료)

수강 전 만들 수 있다면 시간이 단축됩니다. 강의 중에 같이 만들어도 OK입니다.

1. **Google 계정** — Gemini API 발급용
2. **GitHub 계정** — 코드를 인터넷에 보관할 곳 ([github.com](https://github.com))
3. **Supabase 계정** — 데이터베이스 ([supabase.com](https://supabase.com))
4. **Vercel 계정** — 인터넷 배포용 ([vercel.com](https://vercel.com))
5. **Anthropic 계정** — Claude Code 사용용 ([claude.com](https://claude.com))

> 💡 **Tip**: Vercel·Supabase·Anthropic은 **GitHub 계정으로 한 번에 로그인**할 수 있어요. GitHub 계정만 먼저 만들어 두면 나머지가 쉬워집니다.

### 4-3. 비용 안내

본 강의는 **순수 무료**로 진행됩니다.

| 도구 | 우리가 쓰는 범위 | 비용 |

## 5. 예시 서비스 — **"AI 메모장"** (어디까지나 출발점!)

> ⚠️ **중요**: 이 강의에서 공통으로 따라 만드는 **"AI 메모장"은 어디까지나 학습용 예시**입니다.
> 핵심 패턴(회원가입 → 데이터 저장 → AI 활용 → 배포)이 같다면, **수강생 본인이 평소 만들고 싶었던 서비스로 자유롭게 바꿔도 됩니다.**
> 강사는 공통 예시로 시연하지만, 수강생은 자기 아이디어로 바로 응용해 봐도 좋습니다.

### 5-1. 공통 실습 예시 — "AI 메모장"

| 단계 | 기능 |
|---|---|
| 1 | 이메일로 **회원가입 / 로그인** |
| 2 | 내 메모 **작성·조회·수정·삭제** (CRUD) |
| 3 | "**AI로 요약하기**" 버튼 → Gemini가 메모 요약 |
| 4 | 누구나 접속 가능한 **인터넷 URL**로 배포 |

화면 흐름:
```
[로그인 화면] → [내 메모 목록] → [메모 작성/편집] → [AI 요약 버튼]
```

### 5-2. 같은 뼈대로 만들 수 있는 다른 서비스 — **"이걸 해도 됩니다"**

> 💡 아래 예시들은 **모두 강의에서 배운 "회원가입 + DB + AI API + 배포"의 같은 뼈대**로 만들 수 있습니다.
> 강의 시간 안에 본인 아이디어로 바꾸고 싶다면, Claude Code에게 "메모장 대신 X를 만들어 줘"라고 요청하면 됩니다.

| 아이디어 | "메모" 자리에 들어가는 것 | AI가 해 주는 일 | 누가 좋아할까 |
|---|---|---|---|
| 📝 **AI 강의 노트 정리** | 수업 중 적은 거친 노트 | 요점 정리 + 핵심 키워드 추출 | 학생, 강의 듣는 사람 |
| 📖 **AI 독서 기록장** | 책에서 메모한 문장·인용구 | 책별 요약 + 비슷한 책 추천 | 독서가 |
| 🎨 **그림 일기** ⭐ | 오늘의 일기 텍스트 | **일기 → 그림 자동 생성** (Gemini 이미지 생성 무료 티어 활용) | 일기 쓰는 모두 |
| 🍱 **식단 기록 + 코칭** | 오늘 먹은 음식 사진/텍스트 | 칼로리·영양 분석 + 코멘트 | 다이어트·건강관리 |
| 💪 **운동 일지** | 오늘의 운동 종목·세트 | 약점 분석 + 다음 주 추천 루틴 | 헬스·러닝하는 분 |
| 💡 **아이디어 노트** | 떠오른 사업·발명 아이디어 | 보완점 + 실행 단계 제안 | 기획자, 창업 준비생 |
| 🛒 **쇼핑 위시리스트** | 사고 싶은 물건 + 가격 | 우선순위 추천 + 비슷한 상품 검색 | 충동구매 줄이고 싶은 분 |
| 🎬 **영화·드라마 감상록** | 본 작품 + 짧은 감상 | 취향 분석 + 다음 추천작 | 영화 좋아하는 분 |
| 📅 **하루 회고** | 오늘 있었던 일·감정 | 감정 패턴 분석 + 격려 멘트 | 자기개발 관심 있는 분 |
| 🌱 **반려식물 일지** | 식물 상태 메모·물 준 날 | 다음 물 줄 시기 예측 + 케어 팁 | 식물 키우는 분 |

> 🎨 **"그림 일기"가 가능한 이유**: Google Gemini 무료 API는 **텍스트뿐 아니라 이미지 생성도 일부 모델에서 지원**합니다. (`gemini-2.0-flash-image` 계열, 무료 티어 한도 내 사용 가능)
> 본 강의에서는 5차시에서 **메모 요약(텍스트 → 텍스트)**을 메인으로 다루지만, **응용으로 "일기 → 그림 생성"** 변형 예제도 워크북 부록에서 안내합니다.

### 5-3. 강사 안내 멘트 (예시)

> "이 강의의 메인 예시는 'AI 메모장'이지만, 사실 **'메모'라는 단어 자리에 여러분이 원하는 걸 넣으면** 그게 바로 여러분의 서비스가 됩니다.
> 일기를 넣으면 일기 앱, 강의 노트를 넣으면 노트 정리 서비스, 운동 기록을 넣으면 운동 일지가 됩니다.
> Claude Code에게 한국어로 '메모장 대신 일기 앱을 만들고 싶어'라고 말만 하면, 골격은 그대로 두고 단어와 화면만 바꿔서 다시 만들어 줍니다.
> 그러니 **'나는 이 예시 말고 다른 게 만들고 싶다'는 생각이 드는 분은 환영입니다.** 골격은 같습니다."

## 9. 강의 운영 팁 (강사용)

- **"왜 이걸 하는지"를 먼저 설명**합니다. 비개발자는 맥락 없이 "이 명령어를 치세요"라고 하면 길을 잃습니다.
- 전문 용어가 나오면 **즉시 옆에 일상 비유**를 붙입니다 (예: "API = 식당의 주문서").
- 실습은 **"강사 시연 → 수강생이 그대로 따라하기 → 짧은 변형"** 3단 구조로.
- 진도가 늦은 수강생을 위한 **"체크포인트 코드"**를 미리 GitHub에 올려두면 좋습니다 (각 차시 끝난 시점의 완성본).
- AI(Claude Code)가 **다른 답변을 줘도 당황하지 않도록** "이게 정답이 아니라, 여러 답 중 하나"임을 강조합니다.

## 11. 자주 묻는 질문 (FAQ)

**Q. 코딩을 하나도 모르는데 따라갈 수 있나요?**
A. 네. 모든 코드는 Claude Code가 만들어 줍니다. 우리는 "원하는 것을 한국어로 잘 설명하는 법"을 배웁니다.

**Q. Mac은 안 되나요?**
A. 됩니다. 다만 강의는 **Windows 기준**으로 화면·명령어가 통일되어 있고, Mac 차이는 워크북 부록에서 안내합니다.

**Q. 정말 돈이 한 푼도 안 드나요?**
A. 본 강의 범위(개인 학습용 트래픽)에서는 **모든 도구가 무료 한도 내**에 들어옵니다. 단, Claude Code 자체를 본격적으로 쓰려면 Anthropic 유료 플랜을 권장합니다 (무료 체험으로 강의는 충분).

**Q. 만든 서비스를 친구한테 보여줄 수 있나요?**
A. 네. 6차시가 끝나면 `https://내이름-myapp.vercel.app` 같은 **공개 URL**을 받습니다.
